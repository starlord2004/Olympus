# Olympus University Web Portal

Welcome to the **Olympus University** web portal project! This is a comprehensive, production-ready frontend for a fictional premier university located in Bengaluru. It is designed with high-end institutional standards, offering an interactive, visually stunning, and dynamic experience.

## 🚀 Features

- **Multi-Page Architecture:** Includes fully fleshed-out pages such as Home, About, Academics, Departments, Admissions, Placements, Research, Campus Life, Alumni, and Contact.
- **Modern UI/UX & Animations:** Features smooth transitions, scroll-based animations, glassmorphism elements, a dynamic hero section with particles, and a customized loader.
- **AI-Powered Chatbot ("Zeus"):** An integrated, interactive floating chatbot to assist students and visitors.
- **Student & Faculty Portals:** Includes animated, particle-based login interfaces for both students and faculty members.
- **Responsive Design:** Fully mobile-friendly layouts ensuring optimal viewing across all devices.
- **Theme Toggling:** Supports Dark and Light modes for improved accessibility and user preference.

## 🛠️ Technology Stack

- **HTML5:** Semantic and accessible structure.
- **CSS3:** Vanilla CSS tailored for premium aesthetics, modern typography, grid/flexbox layouts, and custom animations.
- **JavaScript (Vanilla):** Dynamic interactions, DOM manipulation, particle effects, and chatbot logic without heavy framework dependencies.

## 📂 Project Structure

- `index.html` - The main landing page.
- `about.html`, `academics.html`, `admissions.html`, `alumni.html`, `campus.html`, `contact.html`, `cse.html`, `departments.html`, `faculty.html`, `placements.html`, `research.html`, `students.html` - Dedicated content pages.
- `styles.css` - Global and page-specific styles.
- `main.js` - Core website interactivity, animations, and top-level logic.
- `login.css` / `login.js` - Styles and logic for the interactive login portals.
- `chatbot.js` - Logic and UI behaviors for the "Zeus" AI chatbot.
- `logo.png` / `campus-hero.png` - Key image assets.

## 💻 Getting Started

There is no complex backend or build process required to view the frontend website. 

1. **Clone or Download** the repository to your local machine.
2. **Open `index.html`** in any modern web browser (Google Chrome, Mozilla Firefox, Safari, Edge).
3. **Explore** the different pages, test the dark mode toggle, and interact with the AI chatbot!
